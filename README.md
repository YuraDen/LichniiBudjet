# Личный бюджет
Desktop-приложение на C# WinForms (.NET Framework 4.7.2)

Функции:
- Добавление, редактирование и удаление информации об операциях по счетам
- Выполнение переводов между счетами
- Формирование и сохранение в pdf-файл выписки по счёту
- Хранение данных в MySQL
- Отображение данных в DataGridView

Технологии:
- C#
- WinForms
- MySQL
